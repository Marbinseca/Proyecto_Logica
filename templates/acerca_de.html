Acerca del Analizador Lógico

Esta aplicación es una herramienta educativa diseñada para ayudar a estudiantes y entusiastas de la lógica a trabajar con lógica proposicional de una manera interactiva y visual.

Características Principales

  • Intérprete Proposicional: Convierte oraciones en lenguaje natural a su forma simbólica utilizando un modelo de IA (Google Gemini) para interpretar la semántica.
  • Generador de Tablas de Verdad: Crea tablas de verdad completas para cualquier fórmula simbólica, clasificándola como tautología, contradicción o contingencia.
  • Simplificador de Fórmulas: Reduce expresiones lógicas complejas a su forma más simple aplicando las leyes del álgebra booleana.

Tecnologías Utilizadas

El proyecto está construido con un stack de tecnologías moderno y robusto:

  • Backend: Python con el micro-framework Flask.
  • Inteligencia Artificial: La API de Google Gemini (modelo Flash) para todas las operaciones lógicas: interpretación de lenguaje natural, generación de tablas de verdad y simplificación de fórmulas.
  • Frontend: HTML, CSS y JavaScript, con el framework Bootstrap 5 para un diseño responsivo y moderno.
